# Тайны старого маяка — Описание проекта

## Общая информация
- **Платформа:** Android
- **Движок:** Unity 2D
- **Жанр:** Квест (point-and-click)
- **Управление:** Тапы и свайпы по экрану
- **Количество уровней:** 3
- **Срок:** 3 дня
- **Бюджет:** 6000 рублей
- **Цель:** Курсовой проект

## Сюжет

Главный герой — молодой историк, получивший в наследство старый маяк на отдалённом острове. Приехав осмотреть имущество, он обнаруживает, что маяк хранит множество тайн: пропажи кораблей в этом районе, загадочные символы на стенах и дневник предыдущего смотрителя, полный странных записей. Игроку предстоит исследовать остров, разгадать головоломки, найти скрытые предметы и раскрыть многовековую тайну, связанную с древним артефактом, который влияет на морские течения и погоду.

## Геймплей

- **Инвентарь:** Простой список иконок внизу экрана (показывает что "в руках")
- **Взаимодействие:** Только тапы по экрану, без перетаскивания
- **Обратная связь:** Если нельзя взаимодействовать — всплывающая подсказка "Чтобы открыть нужно иметь в инвентаре: [предмет]"
- **Без анимаций:** Смена текстур при взаимодействии (закрыто → открыто)

## Прогрессия уровней

### Уровень 1 — Берег
- **Фон:** `bg_level1_shore.png`
- **Предметы для сбора:**
  1. `item_level1_key.png` — ржавый ключ от сундука
  2. `item_level1_map.png` — старая морская карта с разорванными краями
- **Объект взаимодействия:** Ржавый сундук на пляже
  - `obj_level1_chest_closed.png` — закрытый
  - `obj_level1_chest_open.png` — открытый (пустой)
- **Условие перехода:** Открыть сундук ключом → переход на уровень 2

### Уровень 2 — Лес
- **Фон:** `bg_level2_forest.png`
- **Предметы для сбора:**
  1. `item_level2_amulet.png` — светящийся синий амулет с рунами
  2. `item_level2_hatchet.png` — старый топор
- **Объект взаимодействия:** Дверь заброшенной хижины
  - `obj_level2_door_closed.png` — запертая дверь с задвижкой
  - `obj_level2_door_open.png` — открытая дверь
- **Условие перехода:** Открыть дверь топором → переход на уровень 3

### Уровень 3 — Катакомбы (Финал)
- **Фон:** `bg_level3_catacombs.png`
- **Предметы для сбора:**
  1. `item_level3_masterkey.png` — бронзовый мастер-ключ с якорем
  2. `item_level3_tablet.png` — каменная табличка с рунами
- **Объект взаимодействия:** Каменный сундук с золотыми украшениями
  - `obj_level3_chest_closed.png` — закрытый с магическим замком
  - `obj_level3_chest_open.png` — открытый с сияющим письмом внутри
- **Финал:** Открыть сундук мастер-ключом → появляется `item_final_letter.png` → текстовое сообщение о завершении игры

### Меню
- **Фон:** `bg_menu_lighthouse.png` — маяк на закате
